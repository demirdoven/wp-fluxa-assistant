Fluxa eCommerce Assistant WooCommerce

AI assistant for WooCommerce that converts visitors into customers.

Hero Value

  • Conversational AI via Sensay Replica (store‑aware).
  • Conversation tracking into first‑party DB.
  • Event logging: cart, coupon, thank‑you.
  • Admin tools: chat history+pagination, single view.
  • UX nudges: unread badge + title counter.
  • Privacy‑friendly identity (guest→user merge).

Differentiators

  • End‑to‑end data model (conversations, messages, events, feedback).
  • Resilient frontend + defensive DB creation.
  • Admin observability: last_seen/online, message paging.

Credibility

  • Track: rest_track_conversation() → wp_fluxa_conv
  • Chat: /chat, History: /chat/history
  • Events: Fluxa_Event_Tracker → wp_fluxa_conv_events
  • Admin UI: templates/admin-chat-history.php

Routes

  • /wp-json/fluxa/v1/chat
  • /wp-json/fluxa/v1/chat/history
  • /wp-json/fluxa/v1/conversation/track
  • /wp-json/fluxa/v1/feedback
