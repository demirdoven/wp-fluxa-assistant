Fluxa eCommerce Assistant WordPress + WooCommerce

AI-powered storefront chatbot with conversation tracking, admin chat history, feedback, and first‑party event analytics integrated with Sensay.

Simple Feature List

  • Chatbot widget with unread badges and title counter
  • AI chat powered by Sensay Replica
  • Conversation tracking into database
  • Visitor identity tracking (guest → user merge)
  • Chat history fetch and display
  • Feedback/rating on conversations
  • Admin: Chat History with pagination and sorting
  • Admin: Single-conversation view
  • Quickstart onboarding wizard
  • Event tracking for storefront actions
  • WooCommerce integration (cart, coupon, thank you)
  • Knowledge Base training tools
  • REST API for chat, history, tracking, admin data
  • Robust DB schema for conv/messages/events/feedback

Detailed Feature List

  • Modern launcher UI in templates/chatbot-widget.php + styles in assets/css/chatbot.css
  • Resilient chat JS in assets/js/chatbot.js with conversation UUID extraction and tracking
  • REST: /chat, /chat/history, /conversation/track, /feedback under /wp-json/fluxa/v1
  • Admin Chat History UI with loader + API-driven pagination/sorting in templates/admin-chat-history.php
  • Event logging via Fluxa_Event_Tracker to wp_fluxa_conv_events
  • WooCommerce hooks for add/remove cart, update qty, coupon, thank-you
  • DB tables: wp_fluxa_conv, wp_fluxa_messages, wp_fluxa_conv_events, wp_fluxa_feedback
  • Quickstart wizard and admin notices for setup flow
